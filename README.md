## 🔢 Calculator App
A responsive and interactive Calculator Application developed using HTML, CSS, and JavaScript. This project demonstrates core JavaScript concepts, DOM manipulation, event handling, and responsive user interface design.

## 🚀 Live Demo

🔗 https://kashishtech20.github.io/CalculatorApp/

## 📌 Project Overview
The Calculator App allows users to perform basic arithmetic operations through a clean and user-friendly interface. It was built as a JavaScript mini project to strengthen front-end development skills and practical understanding of DOM manipulation.

##✨ Features

* Basic Arithmetic Operations
  * Addition (+)
  * Subtraction (-)
  * Multiplication (*)
  * Division (/)
* Percentage Calculation (%)
* Clear All Functionality (AC)
* Responsive Design
* Interactive User Interface
* Real-Time Input Display

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
